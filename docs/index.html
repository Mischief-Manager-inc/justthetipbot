Available Commands

/balance

Show your balances across all supported cryptocurrencies

/tip @user amount coin

Tip a user (e.g., /tip @bob 0.1 sol)

/registerwallet coin address

Register your external wallet address for a specific coin

/withdraw address amount coin

Withdraw funds to your external wallet

/airdrop amount coin

Create an airdrop for community members to collect

/help

Display help information and command list
