Smart Contract Commands

/register-wallet <address>

Register your Solana wallet address for smart contract interactions

/sc-tip @user <amount>

Create a smart contract tip transaction for the user to sign

/sc-balance

Check your on-chain wallet balance directly from Solana

/generate-pda

Generate your Program Derived Address for advanced features

/sc-info

View smart contract bot information and program details

/help

Display help information and available smart contract commands
